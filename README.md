## 개요

개인적으로 헬스를 오랜기간했는데, 같이 운동할 파트너가 없었습니다.

파트너와 운동하면 정말 좋은 효율을 낼 수 있는데… 헬스장에서 만나는 사람과 운동할 기회는 있지만..연락하며 관계를 유지하는 것에 힘쓰기 싫었습니다.

그리고 저같은 사람이 꽤 있는 것으로 판단되어 만들기 시작하였습니다.
